### 关键技术决策
- 文字定位使用 `text=全部产品分类` 格式（用户确认）
- 强制等待3s用 `page.wait_for_timeout(3000)`
- 滚动到顶部用 `page.evaluate("window.scrollTo(0, 0)")`
- 截图文件名格式：`{PASS|FAIL}_{timestamp}_{testname}.png`

### 框架优化修复（第二轮）
修复了审查中发现的所有问题：
1. **BasePage 新增通用方法**：`locate_any()` / `click_any()` / `wait_for_any_visible()` — 消除各页面对象中的重复容错逻辑
2. **CustomerLoginPage.customer_login() 增加登录成功断言** — 避免未登录就返回 HomePage
3. **所有页面对象的复合选择器已修复** — 原 `", "` 分隔的复合选择器无法工作，改为 `*_ALT` 多选择器列表 + `click_any()`
4. **home_page.py PRODUCT_ECO 定位器修复** — 改为 `["text=生态+非生态", ...]` 更精确匹配产品名称
5. **conftest.py 截图 hook 完善** — 已支持成功/失败均截图
6. **pytest.mark.run 改为 pytest.mark.order** — 需安装 `pytest-order==1.2.0`（已添加到 requirements.txt）
7. **customer_login_page.py 类型注解修复** — 添加 `from __future__ import annotations`，用 `str | None` 替代 `Optional[str]`
