## 密码登录 Tab 切换修复

- **根因**：`text=密码登录` 匹配到 `<span>` 文字节点，不是可点击的 `<p>` 父元素。Playwright `click()` 无法正确触发 Vue `@click` 事件。
- **修复**：
  - 定位器改为 `.login_way_box > p:last-child`（CSS 结构选择器）
  - 点击方式改为 JS `dispatchEvent(new MouseEvent('click', { bubbles: true }))`
  - 点击后 `wait_for_selector("input[type='password']", state="visible")` 确认切换生效
